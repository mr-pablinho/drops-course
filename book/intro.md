# Hydrology Meets Machine Learning

**A free, hands-on course for hydrology researchers and practitioners**

---

Rivers have been measured for centuries. Satellites now observe every corner of the water cycle. And machine learning has fundamentally changed what we can do with that data.

This course teaches you to apply **data science, machine learning, and AI to real hydrological problems** — from predicting streamflow to forecasting drought — using Python, open datasets, and modern tools.

## Who this course is for

You already understand hydrology: discharge, catchments, the water cycle, return periods, drought indices. What you want to learn is **how to bring modern data science to your work**.

No prior Python or ML experience needed.

## What you'll build

By the end, you'll be able to:

- Load and process hydrological time series from USGS, ERA5, and the Caravan dataset
- Build and evaluate machine learning models for streamflow, floods, and drought
- Train LSTM networks for multi-day streamflow forecasting
- Explain *why* a model makes predictions using SHAP
- Think critically about ML model limitations in hydrological contexts

## How to use this course

Every lesson is a **Jupyter notebook** you can run in your browser — no installation required. Click the rocket icon at the top of any page to open it in Google Colab.

```{note}
All code is tested automatically. Every notebook runs end-to-end on clean data before it's published.
```
